# Delegatr — Design Brainstorm

## Idea 1: "Command Center" — Dark Industrial Precision
**Design Movement:** Brutalist-meets-Operator UI (Bloomberg Terminal + Linear)
**Core Principles:**
- High-density information with zero clutter
- Monospace accents for data, sans-serif for prose
- Dark charcoal backgrounds with electric amber/orange highlights
- Grid-based data tables with sharp dividers

**Color Philosophy:** Deep slate (#0F1117) base, electric amber (#F59E0B) for actions, cool gray (#94A3B8) for secondary text. Feels like mission control.

**Layout Paradigm:** Left persistent sidebar with icon+label nav. Main content uses a split-panel: left for input/form, right for live preview/results.

**Signature Elements:**
- Monospace badge labels for roles/skills
- Thin 1px amber accent lines as section dividers
- Terminal-style "parsing" animation

**Interaction Philosophy:** Every action has immediate visual feedback. Parsing feels like a real AI scan.

**Animation:** Staggered fade-in for list items, typewriter effect for AI output, progress bars with pulse.

**Typography System:** JetBrains Mono (data/labels) + DM Sans (body/headings)

## Idea 2: "Clarity OS" — Light Structured Workspace
**Design Movement:** Swiss Grid + Apple Human Interface Guidelines
**Core Principles:**
- Crisp white surfaces with generous whitespace
- Color used sparingly — only for status and action
- Structured hierarchy through size and weight, not decoration
- Cards as the primary unit of information

**Color Philosophy:** Off-white (#FAFAFA) base, deep navy (#1E3A5F) for primary actions, sage green (#22C55E) for success/completion, warm amber for warnings. Professional and trustworthy.

**Layout Paradigm:** Top navigation bar + full-width content area. Wizard-style multi-step flow for task creation. Dashboard uses a 3-column card grid.

**Signature Elements:**
- Pill-shaped skill tags with subtle color coding
- Circular avatar initials with status rings
- Step progress indicator at the top of forms

**Interaction Philosophy:** Guided, wizard-like flows reduce cognitive load. Each step is one focused task.

**Animation:** Smooth slide transitions between steps, card hover lifts, skeleton loading states.

**Typography System:** Sora (headings) + Inter (body) — clean, modern, readable

## Idea 3: "Signal" — Dark Teal Precision Dashboard
**Design Movement:** Data-first SaaS (Vercel + Stripe Dashboard aesthetic)
**Core Principles:**
- Dark background with teal/cyan accent system
- Information hierarchy through luminosity contrast
- Sidebar navigation with collapsible sections
- Micro-typography details for dense data

**Color Philosophy:** Near-black (#0A0E1A) canvas, teal (#14B8A6) as the primary signal color, electric blue (#3B82F6) for secondary actions, muted slate for supporting text. Feels like a high-end analytics tool.

**Layout Paradigm:** Fixed left sidebar (240px) with icon nav. Main area has a top breadcrumb bar. Content uses asymmetric layouts — wide left panel + narrow right metadata panel.

**Signature Elements:**
- Glowing teal ring indicators for active/online status
- Frosted glass cards with subtle border glow
- Animated skill radar/bar charts per employee

**Interaction Philosophy:** Data is always visible. Hover reveals depth. Click reveals action.

**Animation:** Entrance animations with staggered delays, smooth sidebar collapse, chart draw-on animations.

**Typography System:** Space Grotesk (headings) + Geist (mono data) + system-ui (body)

---

## Selected Design: Idea 3 — "Signal"

Dark teal precision dashboard. Space Grotesk + Geist Mono. Near-black canvas with teal accent system. Fixed sidebar, asymmetric layouts, frosted glass cards, animated skill indicators.
